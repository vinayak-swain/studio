# **App Name**: Nebula Dashboard

## Core Features:

- Top Repositories Display: Display a list of top repositories with search and 'New' pills.
- Home Page Prompt Card: Display a prominent 'Ask Anything' prompt to encourage user interaction.
- Content Feed: Show a feed of content cards relevant to the user.
- Changelog Display: Display latest changelog items in a timeline format.
- Repository Suggestions: AI-powered suggestions for adding relevant repositories. A tool to suggest files or spaces for the repositories
- Issue Generation: Ability to create issues from dashboard with action buttons

## Style Guidelines:

- Background color: Dark gradient from #010409 to #0d1117.
- Text color: High-contrast white (#FFFFFF) and gray (#A0A0A0) text for readability. Primary accent color: Blue (#2188FF) for links and highlights.
- Body and headline font: 'Inter', sans-serif font, with medium-weight section titles and smaller meta text.
- Card-based layout with 8-12px border-radius, subtle inner shadows and a 1px darker gray border.
- Global top navbar with logo, navigation, search bar, and utility icons. Three-column layout with left/right sidebars and fluid center content. Generous 16-24px padding inside cards, 12-16px gaps between components. Responsive design with column stacking on smaller widths.
- Subtle hover animations for cards (slight lift/brighten) and buttons (background shift).
